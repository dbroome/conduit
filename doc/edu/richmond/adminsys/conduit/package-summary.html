Package edu.richmond.adminsys.conduit

  • Interface Summary 
    Interface Description
    ICFitting The ICFitting interface is the superinterface for all source, sink, and monitor components in the Conduit framework
    ICMessagePump The ICMessagePump interface is implemented by a class that distributes events from the source and sink objects to the monitor objects.
    ICMonitor The ICMonitor interface is implemented by any classes that will observe the events generated by the source and sink components and display or persist them.
    ICSink The ICSink interface is implemented by any classes that write data to a destination.
    ICSource The ICSource interface is implemented by any classes that read data from a source.
  • Class Summary 
    Class Description
    Conduit This class is the embodiment of a single data transformation.
    ConduitFactory The ConduitFactory class constructs instances of Conduit objects when supplied with GPathConfig objects containing their configuration settings.
    DefaultMessagePump The DefaultMessagePump class is a simple implementation of the ICMessagePump interface.
    Main The Main class contains the entry point invoked by java -jar Conduit.jar conduits.xml.
    Utilities The Utilities class contains static convenience methods for other classes in the framework to use.
  • Exception Summary 
    Exception Description
    ConduitException The ConduitException is the base class for any exceptions thrown by the framework during a transformation.
    ConfigurationException A ConfigurationException is thrown by the framework if there is a problem reading the XML configuration file or setting properties on source, sink, or monitor objects.
